## 项目简介（200字）

AlphaMind Lite 是专为币安新手投资者打造的 7×24 小时智能交易伙伴。通过智能风控哨兵实时监控仓位健康度与爆仓风险，结合社群情报 Agent 追踪公告与新币动态，让用户在复杂市场中获得全天候守护。中文本地化设计，支持 Telegram/QQ 多通道推送。核心差异化：不是"又一个交易机器人"，而是"24小时不眨眼的市场哨兵"。

## 核心功能

1. **🛡️ 风控哨兵**
   - 仓位健康度实时监控
   - 爆仓风险提前预警
   - 一键避险建议

2. **📡 情报 Agent**
   - 币安公告监控 + 中文翻译
   - 新币上线第一时间提醒
   - KOL 喊单监控

## 竞争优势

- 中文本地化（中国用户专属）
- 多 Channel 推送（Telegram/QQ）
- 零门槛自然语言交互
- 基于 OpenClaw 快速迭代

## 技术栈

- OpenClaw Gateway
- Binance API
- Node.js
- Tavily 搜索

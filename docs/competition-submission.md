## 第 2 题：您希望解决什么币安相关的问题？/达成什么目标？

**解决币安普通用户在投资决策中的三大核心痛点：**

**1. 信息碎片化问题**
币安用户需要同时关注行情价格、恐慌贪婪指数、K线走势、成交量等多维数据，这些信息分散在不同页面和工具中。AlphaMind Lite 提供一站式 Dashboard，将 BTC/ETH 实时价格、Fear & Greed 指数、综合买卖信号、K线图表整合到一个页面，打开浏览器即可获得市场全貌。

**2. 杠杆交易风控缺失**
很多币安合约用户不清楚自己的爆仓价格在哪里，经常因为仓位管理不当被强制平仓。AlphaMind Lite 内置杠杆风险计算器，用户只需输入币种、数量、入场价和杠杆倍数，系统自动计算爆仓价格、保证金水平和风险评级（安全/警告/危险），帮助用户在下单前就了解风险。

**3. 投资决策困难**
普通用户面对复杂的市场数据难以判断买卖时机。AlphaMind Lite 内置 AI 智能助手，用户可以用自然语言直接问"现在该买BTC吗？"、"市场风险如何？"，AI 会结合实时行情、恐慌指数、技术趋势给出分析建议。

**产品目标：让每一个币安用户，无论技术水平如何，都能拥有机构级别的投资分析能力。**

**核心亮点：**
- 零代码操作 — 纯 Web 界面，无需任何技术背景
- 零依赖部署 — 纯 Node.js 实现，一条命令启动
- 数据真实 — 直接对接 Binance API，无模拟数据
- 完全开源 — MIT 协议，代码透明可审计

## 第 5 题：在线 Demo 链接 + GitHub 仓库

**GitHub 仓库：** https://github.com/ailin546/alphamind-lite

**本地 Demo 启动方式：**
```bash
git clone https://github.com/ailin546/alphamind-lite.git
cd alphamind-lite
node server.js
# 打开 http://localhost:3000
```

> 如果需要在线 Demo，可以用以下方式快速部署：
> - **Railway**: 连接 GitHub 仓库，设置 Start Command 为 `node server.js`
> - **Render**: 创建 Web Service，Start Command 为 `node server.js`
> - **Replit**: 导入 GitHub 仓库即可运行

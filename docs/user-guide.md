## 📖 功能介绍

### 1. 实时行情监控

AlphaMind Lite 连接币安 API，实时获取主流币种价格数据。支持的币种包括 BTC、ETH、BNB、SOL、DOGE 等热门加密货币。每隔一段时间自动刷新，确保你掌握最新市场动态。界面清晰展示当前价格和 24 小时涨跌幅，用绿涨红跌的颜色直观区分。

### 2. 恐慌指数分析

通过 Alternative.me 获取加密市场的 Fear & Greed Index（恐慌指数），帮助判断市场情绪。指数范围 0-100，数值越低表示市场越恐慌（可能是买入机会），数值越高表示市场越贪婪（可能存在风险）。结合 AI 算法给出买卖建议，辅助你做出更理性的交易决策。

### 3. 投资组合分析

输入你的持仓信息，AlphaMind Lite 自动计算总资产价值、盈亏比例和风险分散度。支持多币种同时管理，直观展示各类资产占比。AI 会根据你的持仓情况给出个性化建议，如建议分散持仓、设置止损线等。

### 4. 情报监控系统

实时抓取币安官方公告，自动翻译成中文推送给用户。支持新币上线提醒、KOL 喊单监控等功能。第一时间获取市场重要信息，消除信息差。在睡觉时也能掌握币安动态，不错过任何重要机会。

### 5. 风控预警系统

输入仓位信息（币种、数量、买入价、杠杆倍数），自动计算距离爆仓线还有多少空间。给出风险评级（安全/警告/危险），并提供一键避险建议。帮助你在睡觉时也能安心，不用担心突然暴跌导致爆仓。

## ❓ 常见问题解答

### Q1: AlphaMind Lite 是免费的吗？

是的，AlphaMind Lite 是开源免费项目。你可以前往 GitHub 仓库克隆并自行部署。部分高级功能（如实时推送）可能需要配置自己的 Telegram Bot 或服务器。

### Q2: 如何开始使用？

```bash
# 克隆项目
git clone https://github.com/ailin546/alphamind-lite.git
cd alphamind-lite

# 运行完整演示
node scripts/comprehensive-demo.js
```

### Q3: 数据来源安全吗？

数据主要来自币安官方 API 和 Alternative.me 等可信数据源。AlphaMind Lite 仅作为信息聚合和展示工具，不存储你的交易密钥或敏感信息。

### Q4: 支持哪些交易平台？

目前主要支持币安（Binance）交易所。未来计划扩展到 OKX、Bybit 等主流交易所。

### Q5: 如何获取实时推送？

需要在 Telegram 创建 Bot 并获取 Token，然后在配置文件中填入即可。详细教程请参考项目 README。

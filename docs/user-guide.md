## 📖 产品概述

AlphaMind Lite 是一款面向普通投资者的加密货币智能助手，旨在帮助用户做出更理性的投资决策。通过整合实时行情、恐慌指数分析、持仓管理和风控预警等功能，让每个普通投资者都能获得机构级的交易智慧。

**核心特点：**
- 🆓 完全免费开源
- 🔒 数据本地处理，保障隐私
- 🤖 AI 驱动的智能分析
- 📱 轻量级，易于部署

## 🛠️ 安装部署

### 环境要求

| 要求 | 最低版本 |
|------|----------|
| Node.js | 18.0.0 |
| npm | 9.0.0 |

### 安装步骤

```bash
# 1. 克隆项目
git clone https://github.com/ailin546/alphamind-lite.git
cd alphamind-lite

# 2. 安装依赖
npm install

# 3. 运行演示（可选）
node scripts/comprehensive-demo.js

# 4. 启动主程序
node index.js
```

### 配置说明

首次使用时，需要在 `config/` 目录下进行必要配置：

```javascript
// config/api.js 示例
module.exports = {
  // 币安 API（可选，仅高级功能需要）
  binance: {
    apiKey: 'your-api-key',
    secretKey: 'your-secret-key'
  },
  
  // Telegram Bot（可选，用于推送）
  telegram: {
    botToken: 'your-bot-token',
    chatId: 'your-chat-id'
  },
  
  // OpenAI API（可选，用于 AI 对话）
  openai: {
    apiKey: 'your-openai-key'
  }
};
```

## 📖 功能介绍

### 1. 实时行情监控

AlphaMind Lite 连接币安 API，实时获取主流币种价格数据。

**支持的币种：**
- 主流币：BTC、ETH、BNB
- 热门币：SOL、DOGE、XRP、ADA
- DeFi 热门：AVAX、DOT、MATIC

**功能特点：**
- 每隔 5 秒自动刷新价格
- 绿涨红跌的颜色直观区分
- 24 小时涨跌幅统计
- 成交量和最高最低价显示

### 2. 恐慌指数分析

通过 Alternative.me 获取加密市场的 Fear & Greed Index（恐慌指数）。

**指数解读：**

| 数值范围 | 情绪等级 | 建议 |
|----------|----------|------|
| 0-25 | 极度恐慌 | 可能是买入机会 |
| 25-45 | 恐慌 | 适合分批建仓 |
| 45-55 | 中性 | 保持观望 |
| 55-75 | 贪婪 | 谨慎追高 |
| 75-100 | 极度贪婪 | 考虑止盈 |

**AI 建议：** 结合历史数据分析当前情绪，给出买卖建议。

### 3. 投资组合分析

输入你的持仓信息，AI 自动进行分析。

**支持功能：**
- 多币种持仓管理
- 总资产价值计算
- 盈亏比例统计
- 风险分散度评估
- 个性化投资建议

**持仓配置示例：**

```javascript
// scripts/portfolio.js
const PORTFOLIO = [
  { symbol: 'BTC', amount: 0.5, buyPrice: 70000 },
  { symbol: 'ETH', amount: 5, buyPrice: 2000 },
  { symbol: 'SOL', amount: 100, buyPrice: 80 }
];
```

### 4. 情报监控系统

实时抓取币安官方公告，自动翻译成中文。

**监控内容：**
- 🚀 新币上线公告
- 📢 平台重要通知
- 👥 KOL 喊单监控
- 📊 市场活动信息

### 5. 风控预警系统

输入仓位信息，智能计算风险。

**输入参数：**
- 币种
- 持仓数量
- 买入价格
- 杠杆倍数（如有）

**输出分析：**
- 距离爆仓线距离
- 风险评级（安全/警告/危险）
- 一键避险建议
- 止损线设置参考

## ❓ 常见问题解答

### Q1: AlphaMind Lite 是免费的吗？

是的，AlphaMind Lite 是开源免费项目。你可以前往 GitHub 仓库克隆并自行部署。部分高级功能（如实时推送）可能需要配置自己的 Telegram Bot 或服务器。

### Q2: 如何开始使用？

```bash
# 克隆项目
git clone https://github.com/ailin546/alphamind-lite.git
cd alphamind-lite

# 运行完整演示
node scripts/comprehensive-demo.js
```

### Q3: 数据来源安全吗？

数据主要来自币安官方 API 和 Alternative.me 等可信数据源。AlphaMind Lite 仅作为信息聚合和展示工具，不存储你的交易密钥或敏感信息。所有 API Key 保存在本地配置中，不会上传任何服务器。

### Q4: 支持哪些交易平台？

目前主要支持币安（Binance）交易所。未来计划扩展到 OKX、Bybit、火币等主流交易所。

### Q5: 如何获取实时推送？

需要在 Telegram 创建 Bot 并获取 Token，然后在配置文件中填入即可。详细教程请参考项目 README。

# AlphaMind Lite 用户指南

## 功能介绍

### 📊 市场行情
- 实时获取 Binance 主流币种价格
- 支持 BTC、ETH、BNB、SOL、DOGE 等
- 显示 24h 涨跌幅

### 🎯 恐慌指数
- 加密市场 Fear & Greed Index
- AI 分析市场情绪
- 提供买卖建议

### 💼 持仓分析
- 支持多币种持仓管理
- 实时计算盈亏
- 风险分散度评估

### 📡 情报监控
- 自动抓取 Binance 公告
- 中文翻译推送
- 新币上线提醒

### 🛡️ 风控预警
- 仓位健康度检测
- 爆仓风险预警
- 一键避险建议

## 使用场景

**场景一：日常看盘**
```
运行: node scripts/comprehensive-demo.js
查看当日市场概况和持仓状态
```

**场景二：买入决策**
```
查看恐慌指数 > 30 考虑分批建仓
恐慌指数 < 75 可适当止盈
```

**场景三：风险检查**
```
运行: node scripts/portfolio.js
查看持仓是否集中，设置止损线
```

## 常见问题

**Q1: 需要 API Key 吗？**
A: 基础功能不需要，行情和恐慌指数使用免费 API。

**Q2: 支持哪些交易所？**
A: 当前支持 Binance，其他交易所陆续添加中。

**Q3: 如何添加新币种？**
A: 修改 portfolio.js 中的 PORTFOLIO 数组。

**Q4: 可以设置价格提醒吗？**
A: 可以通过 alerts.js 设置阈值提醒。

**Q5: 收费吗？**
A: 当前免费版足够日常使用，专业版即将推出。

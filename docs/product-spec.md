# AlphaMind Lite 产品规格文档

## 1. 产品愿景

AlphaMind Lite 是一款零依赖、AI驱动的加密货币投资助手，专为 Binance 用户设计。基于纯 Node.js 构建，无需 npm install，一条命令即可启动。

**核心价值主张：**
- **零依赖**：纯 Node.js 原生模块，无第三方包
- **零配置**：`node server.js` 立即可用
- **智能辅助**：基于实时数据驱动的 AI 交易分析
- **安全透明**：本地运行，开源代码，数据自主可控

**目标用户：**
- Binance 加密货币投资者
- 量化交易初学者
- 需要风险管理工具的杠杆交易者

---

## 2. 用户故事

### 用户故事 1：新手投资者
> 作为一名加密货币新手，我希望有一个一站式仪表盘帮助我了解市场趋势，做出更明智的投资决策。

**验收标准：**
- ✅ 仪表盘展示 BTC/ETH 实时价格、涨跌幅
- ✅ 恐慌贪婪指数直观展示市场情绪
- ✅ AI 聊天助手用自然语言回答投资问题

### 用户故事 2：杠杆交易者
> 作为一名杠杆交易者，我需要清楚知道我的爆仓价格和风险等级。

**验收标准：**
- ✅ 输入杠杆倍数和入场价，即时计算爆仓价
- ✅ 风险等级可视化（安全/警告/危险）
- ✅ 提供降低风险的建议

### 用户故事 3：定投策略者
> 作为一名长期投资者，我想模拟 DCA（定期定额）策略的历史表现。

**验收标准：**
- ✅ 输入每月投入金额和期限
- ✅ 基于当前价格计算预期收益
- ✅ 直观展示投资回报率

**技术栈：**
- **运行时**：Node.js >= 20（零第三方依赖）
- **前端**：原生 HTML5 + CSS3 + Vanilla JavaScript
- **图表**：Chart.js (CDN)
- **数据源**：Binance REST API + Fear & Greed Index API
- **存储**：JSON 文件持久化（原子写入）
- **部署**：Docker + Nginx + PM2

## 5. 部署方式

### 5.1 本地运行（最简方式）
```bash
git clone https://github.com/ailin546/alphamind-lite.git
cd alphamind-lite
node server.js
# 打开 http://localhost:3000
```

### 5.2 Docker 部署（推荐生产环境）
```bash
cp .env.example .env
docker compose up -d
# Nginx 反向代理 → http://localhost:80
```

### 5.3 云端一键部署
- **Railway** / **Render** / **Replit**：连接 GitHub 仓库即可

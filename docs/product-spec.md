# AlphaMind Lite 产品规格文档

## 1. 产品愿景

AlphaMind Lite 是一款轻量级加密货币交易辅助工具，旨在为个人投资者和量化交易爱好者提供低门槛、高效率的交易信号识别与策略回测能力。

**核心价值主张：**
- **简洁高效**：开箱即用，无需复杂配置
- **智能辅助**：基于数据驱动的交易信号分析
- **安全透明**：本地运行，数据自主可控

**目标用户：**
- 加密货币个人投资者
- 量化交易初学者
- 交易策略研究者

---

## 2. 用户故事

### 用户故事 1：新手投资者
> 作为一名加密货币新手，我希望能有一个简单的工具帮助我了解市场趋势，这样我可以做出更明智的投资决策，而不会被复杂的技术分析吓倒。

**验收标准：**
- 用户可以在 5 分钟内完成初始配置
- 仪表盘清晰展示 BTC/ETH 等主流币种的趋势信号
- 提供简洁的买入/卖出建议说明

### 用户故事 2：日内交易者
> 作为一名日内交易者，我需要实时监控多个币种的价格走势和异常波动，以便快速捕捉交易机会。

**验收标准：**
- 支持添加 10+ 个自定义监控币种
- 价格异动时推送即时通知
- 图表展示 15 分钟/1 小时/4 小时多周期数据

### 用户故事 3：量化研究员
> 作为一名量化交易研究员，我需要对自己的交易策略进行历史回测，验证策略的有效性，并优化参数设置。

**验收标准：**
- 支持导入自定义交易策略代码
- 提供至少 1 年的历史数据回测
- 输出详细的策略表现报告（收益率、最大回撤、夏普比率等）

### 用户故事 4：技术分析师
> 作为一名技术分析师，我希望能够自定义技术指标组合，创建个性化的信号提醒规则，提高分析效率。

**验收标准：**
- 支持 RSI、MACD、EMA、布林带等常用指标
- 可配置指标参数和阈值
- 支持多指标组合条件触发提醒

## 3. 功能列表详细说明

### 3.1 核心功能

| 功能模块 | 功能名称 | 功能描述 |
|---------|---------|---------|
| **市场数据** | 实时行情 | 接入 Binance API 获取实时价格、成交量、订单簿数据 |
| | 历史数据 | 提供分钟/小时/日线级别历史K线数据 |
| | 多币种监控 | 支持同时监控多个交易对 |
| **信号分析** | 技术指标 | 内置 RSI、MACD、EMA、布林带、KDJ 等指标 |
| | 信号生成 | 基于指标组合自动生成买入/卖出信号 |
| | 异动预警 | 价格波动超过阈值时触发告警 |
| **策略回测** | 策略编辑 | 支持 Python 编写自定义交易策略 |
| | 回测引擎 | 基于历史数据模拟交易执行 |
| | 绩效分析 | 计算收益率、夏普比率、最大回撤等指标 |
| **可视化** | 图表展示 | K线图叠加技术指标 |
| | 仪表盘 | 组合展示多个监控币种的关键信息 |
| | 报告导出 | 支持导出回测报告为 PDF/CSV |

### 3.2 扩展功能

| 功能模块 | 功能名称 | 功能描述 |
|---------|---------|---------|
| **通知系统** | 邮件通知 | 交易信号/异动告警通过邮件推送 |
| | Telegram/Discord | 支持多平台机器人消息推送 |
| **数据存储** | SQLite | 本地轻量级数据库存储 |
| | 数据导出 | 支持历史数据导出为 CSV |

**架构说明：**

- **数据层**：Binance API 获取实时数据，SQLite 本地存储，Redis 缓存加速

## 5. 部署方式

### 5.1 本地部署（开发/测试）

**前置要求：**
- Python 3.9+
- pip 或 poetry

**快速启动：**
```bash
# 克隆项目
git clone https://github.com/your-repo/alphamind-lite.git
cd alphamind-lite

# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入 Binance API Key

# 启动服务
python -m alphamind.main
```

### 5.2 Docker 部署（推荐）

**前置要求：**
- Docker 20.10+
- Docker Compose

**快速启动：**
```bash
# 克隆项目
git clone https://github.com/your-repo/alphamind-lite.git
cd alphamind-lite

# 配置环境变量
cp .env.example .env

# 启动所有服务
docker-compose up -d
```

**Docker Compose 服务组成：**
| 服务 | 端口 | 说明 |
|-----|-----|------|
| web | 8080 | Web UI 和 API 服务 |
| worker | - | 后台任务 worker |
| redis | 6379 | 缓存服务 |
| sqlite | - | 数据持久化 |

### 5.3 云端部署

**支持的平台：**
- **Railway**：一键部署，配置环境变量即可
- **Render**：支持 Docker 部署
- **VPS (Ubuntu)**：使用 Docker Compose 或直接安装

**推荐配置：**
- 1 CPU / 1GB RAM 即可运行
- 10GB SSD 存储

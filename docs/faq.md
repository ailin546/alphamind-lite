# 常见问题

## 基础问题

### 1. 这是什么产品？
AlphaMind Lite 是一个免费的加密货币投资助手，基于 AI 帮助普通投资者做出更好的投资决策。通过整合实时行情、恐慌指数分析、持仓管理和风控预警等功能，让每个普通投资者都能获得机构级的交易智慧。

### 2. 需要付费吗？
基础功能完全免费。高级功能（如自定义 API、更多提醒、自定义交易对）将陆续推出。

### 3. 数据安全吗？
所有数据仅保存在本地，不上传任何服务器。API Key 也仅保存在你的本地配置中，不会上传至任何第三方服务器。

### 4. 支持哪些交易所？
目前支持 Binance（币安），后续将支持 OKX、Bybit、火币等主流交易所。

### 5. 如何开始使用？
只需克隆项目并运行：
```bash
git clone https://github.com/ailin546/alphamind-lite.git
cd alphamind-lite
node scripts/comprehensive-demo.js
```

## 技术问题

### 6. 需要编程知识吗？
不需要。所有功能都可通过简单的命令运行。配置文件也有详细的中文注释，按照示例填写即可。

### 7. 支持哪些操作系统？
支持 Windows、macOS、Linux 系统。只需安装 Node.js 18.0.0 或更高版本即可运行。

### 8. 为什么行情数据不更新？
请检查：
1. 网络是否正常连接
2. 币安 API 是否可访问（部分地区可能被墙）
3. 是否超过 API 请求限制

### 9. 如何查看日志？
运行程序时日志会直接输出到终端。可以使用 `node index.js > log.txt 2>&1` 将日志保存到文件。

## 功能问题

### 10. AI 对话如何配置？
编辑 `scripts/user-ai-chat.js`，填入你自己的 OpenAI API Key 即可。也可以使用其他兼容的 AI API。

### 11. 可以自定义持仓吗？
可以，编辑 `scripts/portfolio.js` 中的 PORTFOLIO 数组即可添加或修改持仓信息。

### 12. 有 Telegram 推送吗？
是的，配置好 Telegram Bot 后即可接收实时推送，包括行情提醒、爆仓预警、公告通知等。

### 13. 恐慌指数是什么？
恐慌指数（Fear & Greed Index）是衡量市场情绪的指标，数值 0-100，越低表示市场越恐慌（可能是买入机会），越高表示市场越贪婪（可能存在风险）。

### 14. 风控预警准吗？
风控预警基于币安官方 API 的实时数据进行计算。但由于加密货币市场波动剧烈，建议作为参考而非绝对依据，始终保持风险意识。

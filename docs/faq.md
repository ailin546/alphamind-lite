# 常见问题

## 1. 这是什么产品？
AlphaMind Lite 是一个免费的加密货币投资助手，基于 AI 帮助普通投资者做出更好的决策。

## 2. 需要付费吗？
基础功能完全免费。高级功能（如自定义 API、更多提醒）将陆续推出。

## 3. 数据安全吗？
所有数据仅保存在本地，不上传任何服务器。API Key 也仅保存在你的本地配置中。

## 4. 支持哪些交易所？
目前支持 Binance，后续将支持更多交易所。

## 5. 如何使用？
只需克隆项目并运行 `node scripts/demo.js` 即可开始体验。

## 6. 需要编程知识吗？
不需要。所有功能都可通过简单的命令运行。

## 7. AI 对话如何配置？
编辑 `scripts/user-ai-chat.js`，填入你自己的 API Key 即可。

## 8. 可以自定义持仓吗？
可以，编辑 `scripts/portfolio.js` 中的 PORTFOLIO 数组即可。

## 9. 有 Telegram 推送吗？
是的，配置好 Bot 即可接收实时推送。

## 10. 如何反馈问题？
欢迎在 GitHub 提交 Issue。

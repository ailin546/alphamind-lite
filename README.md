# AlphaMind Lite - Binance AI Agent Contest

## 项目简介

AlphaMind Lite 是专为币安用户打造的 7×24 小时智能交易伙伴。

## 核心功能

### 🛡️ 风控哨兵
- 仓位健康度实时监控
- 爆仓风险提前预警
- 一键避险建议

### 📡 情报 Agent
- 币安公告监控 + 中文翻译
- 新币上线第一时间提醒
- KOL 喊单监控

## 技术栈

- OpenClaw Gateway
- Binance API
- Node.js

## 快速开始

```bash
# 安装依赖
npm install

# 运行风险监控
node scripts/position-risk.js --example

# 运行新闻监控
node scripts/news-monitor.js
```

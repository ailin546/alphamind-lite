## ✨ 核心功能

### 📊 实时行情监控
毫秒级拉取 Binance 主流币种价格与涨跌幅，告别手动刷盘。一目了然的行情面板，让你随时掌握市场脉搏。

### 🎯 智能恐慌指数
整合 Fear & Greed Index 与 AI 深度分析，不只是告诉你"恐惧"或"贪婪"，更给出具体可执行的操作建议。

### 💼 持仓智能分析
自动追踪多币种盈亏状态，智能计算风险敞口。你的数字资产健康状况，一眼便知。

### 📡 情报聚合雷达
7×24 小时监控币安公告、行业新闻、KOL 信号，重要信息秒级推送。不再错过任何影响市场的关键消息。

### 🛡️ 风控预警系统
实时仓位健康检测，提前预警爆仓风险。让你在风暴来临前做好准备，守护本金安全。

### 🔔 智能价格提醒
自定义价格阈值，Telegram 实时推送。睡觉也能盯盘，不错过任何入场或离场时机。

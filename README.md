## 🎯 项目简介

AlphaMind Lite 是专为币安用户打造的 AI 交易助手，通过人工智能技术帮助普通投资者实现机构级的交易决策。

## ✨ 核心功能

### 📊 实时行情监控
- 主流币种实时价格
- 24h 涨跌幅
- 多维度市场数据

### 🎯 恐慌指数分析
- 加密市场 Fear & Greed Index
- AI 驱动的买卖建议
- 历史数据参考

### 💼 投资组合分析
- 多币种持仓管理
- 盈亏实时计算
- 风险分散度评估
- 个性化投资建议

### 📡 情报监控系统
- 币安公告实时抓取
- 中文翻译推送
- 新币上线提醒
- KOL 喊单监控

### 🛡️ 风控预警
- 仓位健康度检测
- 爆仓风险预警
- 一键避险建议
- 个性化风控策略

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/ailin546/alphamind-lite.git
cd alphamind-lite

# 运行完整演示
node scripts/comprehensive-demo.js

# 单独功能
node scripts/portfolio.js      # 持仓分析
node scripts/fear-greed.js     # 恐慌指数
node scripts/demo-runner.js     # 基础演示
```

##栈

- **Runtime**: OpenClaw Gateway
- **Data**: Binance API, Tavily Search, Alternative.me
- **Language**: Node.js
- **Channels**: Telegram, QQ (可选)
